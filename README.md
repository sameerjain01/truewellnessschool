# True Wellness School

A free, open static website for **True Wellness School** — a holistic wellness brand covering Ayurveda, nutrition, mindfulness, yoga, and brain health.

**Live site:** https://sameerjain01.github.io/truewellnessschool/
**Instagram:** [@truewellnessschool](https://www.instagram.com/truewellnessschool)

---

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero, philosophy, section cards |
| `pillars.html` | The 6 Pillars of True Wellness (Dr. Frank Lipman framework) |
| `programs.html` | Wellness programs & booking form |
| `recipes.html` | The Wellness Kitchen — recipes & nutrition |
| `wisdom.html` | Quotes, affirmations & book recommendations |

## Tech Stack

- Pure HTML5 — no frameworks, no build step
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) via Google Fonts
- Images from [Unsplash](https://unsplash.com) (served via their CDN)
- Hosted on [GitHub Pages](https://pages.github.com/) — free
